Estadísticas

Precision/recall/F1

El algoritmo realiza una clasificación en 3 clases, con una media de precision del 95%



Matriz de confusión

Utilizando una base de datos con mas de 10.000 radiografías, el algorimo es capaz de predecir la mayoría de los casos de forma correcta.

Algoritmo

Utilizando técnicas de Transfer Learning, tomamos como modelo base la red neuronal pre-entrenada de MobileNet, una arquitectura propuesta por Google diseñada para trabajar en entornos con poca capacidad computacional.


La arquitectura original MobileNet :





Utilizaremos esta arquitectura entrenada con ImageNet junto con algunas capas custom creadas para el proceso. Entre estas capas (que pueden ser observadas en la imagen a la izquierda) cabe destacar el uso del dropout para disminuir el sobreajuste.

Trabajo futuro

LIME

Como parte del proceso de mejora, la integración de una explicación LIME ayudaría a la explicabilidad de las imagenes. Todo esto apoyado por una mayor cantidad de imagenes haría el proceso mucho mas eficaz.



Ensembles

Con una mayor capacidad de computo se podría llegar a realizar un ensemble de modelos. Esta técnica permite tratar a la unión de todos los modelos como uno solo.
